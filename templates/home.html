Text Clusters - Home

Text Clusters takes a column of text from a Google spreadsheet and runs the GAA clustering algorithm on it. The cluster tags are then put into the Google Spreadsheet.

The user selects the columns to use, so it can be run iteratively across a number of columns in the same spreadsheet.
